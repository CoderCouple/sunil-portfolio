# Sunil Tiwari — Personal Site

Personal portfolio for Sunil Tiwari. Built with Next.js 15, React 19, Tailwind CSS v4, and Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Editing content

Almost all site content lives in **`app/data.ts`** (projects, work, social links, speaking, books, news, etc.). Judging entries live in `app/judging/judging-data.ts`. Blog index posts are entries in `BLOG_POSTS`; long-form MDX posts go in `app/blog/<slug>/page.mdx`.

## Deployment

Deployed on Vercel at `suniltiwari.io`.
